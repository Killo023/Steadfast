# Images Directory

Place your image files here with these exact filenames:

- **hero.jpg** (or .png, .webp) - Hero section background (recommended: 1920x1080px)
- **handgun.jpg** - Handgun image for pricing section (recommended: 800x600px)
- **training.jpg** - Training image for additional services (recommended: 800x450px)
- **rifle.jpg** - Rifle image for contact section (recommended: 800x450px)
- **security.jpg** - Security image for additional services (recommended: 800x450px)
- **target.jpg** - Target image for trust signals section (recommended: 560x320px)

## Supported Formats
- `.jpg` / `.jpeg`
- `.png`
- `.webp`

The code will automatically use these relative paths. If an image is missing, a neutral gradient fallback will display.
